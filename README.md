# Digital Image Processing (PCD Praktikum)

Digital Image Processing practicum project using Python.

## Requirements

- Python `3.11`
- `pip` (included with Python installation)
- Dependency file: `requirements.txt`

## Setup

Run from the project root:

```bash
git clone https://github.com/Itsjustrell/Digital-Image-Processing.git
cd Digital-Image-Processing/
python3.11 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

## Notes

- Make sure the virtual environment is active (`(.venv)` appears in the terminal) before running scripts.
- If you get dependency errors, reinstall with:

```bash
pip install -r requirements.txt
```
